## Chisai

Chisai is a url shortener built using [hono](https://hono.dev) and Cloudflare [D1](https://developers.cloudflare.com/d1), and deployed on a Cloudflare Worker.

### Getting started

It requires [bun](https://bun.sh) v1.x.x

Launch the server

```sh
$ bun run dev
```

Deploy to Worker

```sh
$ bun run deploy
```
